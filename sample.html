Let's get started building amazing presentation with RevealJS

RevealJS lets you build interactive presentations using HTML + CSS + JavaScript


Step 1

Download the .zip file from my RevealJS Github repo & unzip the folder to your desktop

Download Reveal.js


Step 2

Navigate to your desktop & double-click on the index.html file located inside the unzipped reveal folder

You should see a sample presentation already created that looks something like this:



Step 3

In your favorite text editor (Atom, Sublime Text, etc), open the reveal/ folder located on your desktop



Step 4

Inside your text editor you should see all the file names of the reveal/ folder on the left hand side of your screen



Step 5

Now that we have the files we need, create a new file & save it as an .html file. Go to File > New File > & save it as presentation.html



Step 6

Now that we have a blank presenation.html file saved we can add some basic html elements. Add the following elements & save the file.
