    Let's get started building amazing presentation with Reveal.js

    Reveal.js lets you build interactive presentations using html + css + javascript

    Step 1

    Download the Reveal.js CSS & JavaScript files to your desktop

    Download Reveal.js

    Step 2

    In your favorite text editor (Atom, Sublime Text, etc), open the reveal.js folder located on your desktop

    Step 3

    Inside your text editor you should see all the file names of the reveal.js folder on the left hand side of your screen

    Step 4

    Now that we have the files we need, create a new file & save it as an .html file. Go to File > New File > & save it as index.html

    Step 5

    Now that we have a blank .html file saved we can add the basic html elements to our index.html file

    Add the following elements to your index.html file & save
